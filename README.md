# 🎭 Real-Time Emotion Detection App

A lightweight and efficient web application that uses Artificial Intelligence to detect human emotions in real-time using a webcam.

## ✨ Features
*   **Real-time Detection:** Analyzes facial expressions instantly.
*   **AI-Powered:** Uses `face-api.js` (built on TensorFlow.js).
*   **Minimalist UI:** Built with React and styled for simplicity.
*   **Multiple Emotions:** Detects neutral, happy, sad, angry, surprised, etc.

## 🛠️ Tech Stack
*   **Frontend:** React (Vite)
*   **Library:** [face-api.js](https://github.com)
*   **Runtime:** Node.js
